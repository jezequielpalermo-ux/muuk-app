Base de datos en la nube (Supabase)

  1. Entrá a supabase.com y creá una cuenta gratuita
  2. Creá un nuevo proyecto (elegí un nombre, ej: muuk-flujo)
  3. En el menú izquierdo andá a SQL Editor y ejecutá este código:
CREATE TABLE flujo_caja_state (
  id INTEGER PRIMARY KEY DEFAULT 1,
  data JSONB NOT NULL,
  updated_at TIMESTAMPTZ DEFAULT NOW(),
  updated_by TEXT DEFAULT ''
);
INSERT INTO flujo_caja_state (id, data) VALUES (1, '{}');
ALTER TABLE flujo_caja_state ENABLE ROW LEVEL SECURITY;
CREATE POLICY "acceso total" ON flujo_caja_state FOR ALL USING (true) WITH CHECK (true);

CREATE TABLE flujo_caja_versions (
  id BIGSERIAL PRIMARY KEY,
  data JSONB NOT NULL,
  saved_at TIMESTAMPTZ DEFAULT NOW(),
  saved_by TEXT DEFAULT ''
);
ALTER TABLE flujo_caja_versions ENABLE ROW LEVEL SECURITY;
CREATE POLICY "acceso total" ON flujo_caja_versions FOR ALL USING (true) WITH CHECK (true);
  1. Andá a Settings → API y copiá:
    — Project URL (ej: https://abcd1234.supabase.co)
    — anon / public key (la clave larga que empieza con eyJ)
  2. Mandáselas a Jonathan para que las pegue en el HTML (en las líneas SB_URL y SB_KEY)
Backup y Restauración

Además de Supabase, podés descargar un backup JSON en cualquier momento. El archivo contiene todos los datos y puede restaurarse en caso de emergencia.

El backup se descarga como archivo .json — guardalo en Drive o una carpeta segura.
WhatsApp Bot
Paso 1 — Crear app en Meta for Developers
  1. Andá a developers.facebook.com
  2. Iniciá sesión con tu cuenta de Facebook
  3. Hacé clic en "Mis apps" → "Crear app"
  4. Seleccioná "Otro" como tipo → "Business" como categoría
  5. Nombre: MUUK Bot → clic en "Crear app"
Paso 2 — Activar WhatsApp
  1. En el dashboard de la app, buscá "WhatsApp" y hacé clic en "Configurar"
  2. Seleccioná o creá una cuenta Business
  3. Meta te da un número de prueba gratis (o podés vincular uno propio después)
  4. En WhatsApp → Configuración de la API copiá:
    — Phone Number ID (número largo)
    — Token de acceso temporal (empieza con EAA...)
Paso 3 — Deploy del webhook en Supabase
  1. Instalá el CLI de Supabase: npm install -g supabase
  2. Logueate: supabase login
  3. Descargá el archivo del bot (link abajo) y hacé deploy: 
    supabase functions deploy whatsapp-webhook --project-ref qvfmexkcyrvyidugnaoo
  4. Configurá las variables de entorno: 
    supabase secrets set WA_ACCESS_TOKEN=EAA... WA_PHONE_ID=123... WA_VERIFY_TOKEN=muuk-bot-verify-2026 --project-ref qvfmexkcyrvyidugnaoo
Paso 4 — Conectar el webhook a Meta
  1. La URL del webhook es: 
    https://qvfmexkcyrvyidugnaoo.supabase.co/functions/v1/whatsapp-webhook
  2. En Meta → WhatsApp → Configuración → Webhooks
  3. Hacé clic en "Editar" y pegá la URL del webhook
  4. Token de verificación: muuk-bot-verify-2026
  5. Suscribite al campo: messages
  6. ¡Listo! Mandá "hola" al número del bot para probar
Comandos del bot: resumen, saldos, comisiones, cheques, vencimientos, facturas, km, semanal, debe [empresa]
Cargar facturas por PDF: Mandá un PDF de factura AFIP al bot. El bot lee el contenido, extrae los datos (tipo, N°, CUIT, importes) y te pregunta si es a cobrar o a pagar. Podés escribir "cobrar" o "pagar" como texto del archivo para saltear la pregunta.
Gratis: WhatsApp Cloud API incluye 1.000 conversaciones/mes sin costo. El bot funciona las 24hs desde Supabase Edge Functions.
